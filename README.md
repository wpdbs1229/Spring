# Spring
Spring 공부 repository!

## 삽질
순수 자바프로젝트로 삽질해보기! Spring을 만들어보자!

----
### 순서
1. 순수 자바로 된 프로젝트 생성
2. 직접 심플 스프링 만들기
3. 스프링을 활용해 순수 자바 프로젝트 동작해보기

### 프로젝트 설명
- 편의점 결제 서비스
  - 편의점 간편결제 시스템
  - 사용자 - 편의점 결제 서비스(구현할 서비스) - 결제수단 시스템(제작되어진 시스템이용)

### 프로젝트 환경
- InteliJ
- JAVA 11
- Gradle
- Spring Boot 2.7.9

### 요구사항
- G24, SeeU, 11
- 머니 결제기능 추가

### 클래스
- 결제 서비스
  - 결제
  - 결제 취소
- 머니 어댑터
  - 머니 사용
  - 머니 사용 취소
